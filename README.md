# JS-BASICS Javascript Basics


## JavaScript Arithmetic Operators
|Operator  | Description                    | 
|--------- | -------------------------------
|     +    | Addition                       |
|        - | Subtraction                    |
|        * | Multiplication                 |
|       ** | Exponentiation (ES2016)        |
|        / | Division                       |
|        % | Modulus (Division Remainder)   |
|      ++  | Increment                      |
|      --  | Decrement                      |


### JavaScript Assignment Operators
|Operator |	Example |	Same As
----------|---------|------------|
|  =      | x = y	  | x = y      |
|   +=    | x += y	| x = x + y  |
|   -=	  | x -= y	| x = x - y  |
|   *=	  | x *= y	| x = x * y  |
|   /=	  | x /= y	| x = x / y  |
|   %=	  | x %= y  | x = x % y  |
|   **=   | x **= y | x = x ** y |
